# Odoo Design Exercise

This project involves converting a provided design mock-up into a high-quality, responsive
static landing page. The primary focus is on clean architecture, the use of Bootstrap 5.3
utilities, and a modular SCSS structure.

## Build / run

```bash
# 1. Install build tooling (Bootstrap sources + Dart Sass)
npm install

# 2. Compile SCSS -> css/custom.css
npm run build:css        # one-shot
npm run watch:css        # watch mode while developing

# 3. Serve locally on http://localhost:8080
npm run serve            # uses npx http-server
# or simply open index.html in a browser
```
